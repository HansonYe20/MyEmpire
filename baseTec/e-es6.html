1.可以直接使用cli命令
babel-cli
npm install -g babel-cli
npm install babel-cli --save-dev -g, 属于开发依赖

2.Babel支持很多语法的转码，比如我们想要将ES6转换成ES5，那么需要安装babel-preset-es2015包，如果我们想要编译React源码，就需要安装babel-preset-react，这里我们需要安装babel-preset-es2015
npm install babel-preset-es2015 --save-dev

3.babel baseTec/es6.js --out-file baseTec/es5/es6.js --presets es2015

4.也可以编译整个目录的源文件，只需指定“–out-dir”参数即可

5.直接配置到npm scripts中, 见命令: es6Toes5

6.实时刷新页面: gulp

7.实时刷新页面: webpack
